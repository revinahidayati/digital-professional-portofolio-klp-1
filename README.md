Digital Professional Portfolio

Deskripsi Proyek

Digital Professional Portfolio adalah sebuah proyek yang bertujuan untuk
menampilkan profil profesional secara digital. Portofolio ini memuat
informasi mengenai identitas, keahlian, pengalaman, pendidikan, serta
hasil karya anggota tim dalam bentuk media digital yang terstruktur dan
mudah diakses.

Proyek ini dibuat sebagai bagian dari tugas kelompok untuk meningkatkan
kemampuan kolaborasi, dokumentasi, dan presentasi profesional di bidang
teknologi dan manajemen informasi.


Tujuan Proyek
- Menyajikan portofolio profesional dalam format digital
- Melatih kerja tim dan pembagian peran yang jelas
- Meningkatkan kemampuan dokumentasi proyek
- Menjadi media presentasi profesional yang siap digunakan


Fitur Utama
- Profil anggota tim
- Riwayat pendidikan dan pengalaman
- Keahlian dan kompetensi
- Dokumentasi hasil kerja
- Tampilan sederhana dan mudah dipahami


Teknologi yang Digunakan
- HTML
- CSS
- JavaScript
- Microsoft Word / PDF (dokumentasi)
- GitHub / Media penyimpanan digital
